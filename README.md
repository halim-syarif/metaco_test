# metaco_test

> BackEnd Test

```
$ cd backend
$ npm run migrate-test
$ npm run test

```

<br>

> How To run on localhost

```
$ cd backend
$ npm install
$ npm run migrate
$ npm run seed
$ npm run dev

server listening on port 3000


$ cd ../frontend

note : if backend server not running on port 3000 set port at /src/store/config.js

$ npm install
$ npm run start

```
